GUI Calculator

This program is a GUI based calculator with basic functions (+, -, *, /) made using Python, you can do basic maths with this program. Included within the repository is an exe, a readme file, a license file and a python file. The Python modules I used to create this program are Tkinter and the inbuilt Sys libraries.

To get the .py file working, you need to download Python 3 or above. To download Python go to the Python web page and click the yellow button under 'Download the latest version for Windows' or select your OS under the yellow button. If you don't want to use the Python file or you don't want to install Python, you can simply use the exe by running the included file in the zip file that you can download from one of the two buttons above.
