Flappy Bird

This project is a clone of the popular Flappy Bird. It has a scoring system and menu screen. Included within the downloadable zip file is a readme file, a license file, a zip file of all the assets and a python file. I am not taking credit for designing the graphics and/or gameplay. This is just a project that I wanted to challenge myself with. It utilises the Pygame and Random libraries to generate the gameplay, import images and create the window. For the Python file to read the images, it has to be in the same directory as the Assets folder.

To get the .py file working, you need to download Python 3 or above. To download Python go to the Python web page and click the yellow button under 'Download the latest version for Windows' or select your OS under the yellow button.
