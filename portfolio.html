Welcome to My Portfolio Page

This is a collection of my past projects I have worked on as well as projects I am currently working on. It includes electronics, 3D modelling, PCB design, website design and programming.

R2D2

This repository is for my mini remote control R2D2 project. It includes all the stl files, all the code files, and the f3z (Fusion 360) file. It also includes some Fusion 360 renders and some extra images as well as instructions. The purpose of this project was to test my skills at 3D modelling and PCB and schematic design. I have not yet finished 3D printing all the parts and haven't tested the PCBs or developed the software for R2D2. This is a continuing project

GUI-Based Calculator

This program is a GUI based calculator with basic functions (+, -, *, /) so that you can do basic maths with this program. Included within the repository is an exe, a readme file, a license file and a python file. The purpose of this project was to expand my skills with Python and Tkinter. This is a finished project.

Steel Solutions Consulting

This is a website I designed and developed for Steel Solutions Consulting. It has multiple pages, links to social media, images, navigation menu and drop down menus, an animated slideshow and a contact form using PHP.

Flappy Bird

This program is a clone of the popular Flappy Bird. It has a scoring system and menu screen. Included within the repository is a readme file, a license file and a python file. I am not taking credit for designing the graphics and/or gameplay. This is just a project that I wanted to challenge myself with. This is a finished project.
