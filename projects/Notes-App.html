GUI Text Editor

This is a text editor that will hopefully be close in functionality to Microsoft Word.

It currently has buttons for bold, italic and underline, changing the font size (proof of concept), it also has a toolbar at the top of thd window for opening and saving files, and preferences (proof of concept), an edit menu that has same functionality of bold, italic and underline buttons, undo and redo buttons (proof of concept), and cut, copy and paste buttons (proof of concept). It also has a View menu that curretnly only has the option for dark mode.

Included within the repository is a copy of this readme file, a license file, two python files and, in future, an exectuable file.

The modules it uses from the Tkinter library are:

  • 1. filedialog to ask the user to locate a file to import into the program and to ask the user where to save a file
  • 2. messagebox to alert the user that the program has saved the open file to a location chosen by the user
  • 3. Font to add styles to highlighted text
